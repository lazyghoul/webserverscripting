Web Server Scripting Project

Access to HE (IT)

A simple website that was created
to show the use of specified elements.

Included are the following: Guestbook, Multiuser login, system data checker, and file uploader.

Guestbook

Although the guestbook is not a requirement of the assignment it is still included, this is because it was through the guestbook that I learnt how to create a database with MySQL and have it update with the use of PHP.

Multiuser Login

A login system has been created through the use of PHP and a MySQL database. When a login attempt is made the PHP script will refer to the database to check if the credentials are valid.Also, an Admin account is used to add new users through the use of further PHP.

System Data Checker

One of the criteria for this assignment is the ability to check the Browser and screen resolution of the user. This has been done with the System Data Checker.

MySQL Databases

There are two features of the website that require the storage of data, so two databases are used. One has been used for the user accounts, and the other for the guestbook entries. The PHP script of these two feature incorporate MySQL to read and write to and from these databases.

Site Creation

Along with learning PHP and further expanding my web devolpment skills, I also learned the use of Git. I used GitHub along with Git learning how to clone, commit, push, branch and merge. Even though these skills don't relate to the assignment, I believe they are skills that will serve me well in the future with other projects.

UI/UX

User Interface and User Experience are important to the funcionality of any website. Because of this, I have kept to a simplistic design with everything being accesible from through the header.
